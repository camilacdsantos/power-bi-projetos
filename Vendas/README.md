# 📊 Dashboard de Vendas — Power BI

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um dashboard de análise de vendas utilizando Power BI, com o objetivo de gerar insights sobre a performance comercial e apoiar a tomada de decisão.

Os dados utilizados são do dataset público Sample Superstore Dataset.

---

## 🎯 Objetivos

* Analisar a performance de vendas ao longo do tempo
* Identificar produtos e categorias mais relevantes
* Avaliar distribuição geográfica das vendas
* Monitorar indicadores-chave do negócio

---

## 🧱 Estrutura do Projeto

### 🔹 Modelagem de Dados

* Estrutura em esquema estrela
* Separação entre tabela fato e dimensões
* Criação de tabela calendário

---

### 🔹 Métricas (DAX)

* Receita Total
* Lucro
* Margem
* Ticket Médio
* Crescimento MoM

---

### 🔹 Dashboard

#### 📄 Página 1 — Resumo Executivo

* KPIs principais
* Evolução da receita
* Vendas por categoria
* Vendas por região
* Top produtos

#### 📄 Página 2 — Análise Detalhada

* Impacto de desconto no lucro
* Segmentação de clientes
* Sazonalidade
* Tabela detalhada

---

## 📊 Principais Insights

* Identificação de regiões com maior volume de vendas
* Concentração de receita em determinados produtos
* Impacto de descontos na lucratividade
* Tendências de crescimento ao longo do tempo

---

## 🛠️ Tecnologias Utilizadas

* Power BI

* Power Query
