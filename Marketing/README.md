# 📊 Dashboard de Marketing — Power BI

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um dashboard de análise de marketing utilizando Power BI, com foco na avaliação de campanhas e comportamento de clientes.

Os dados foram obtidos a partir de um dataset público de campanhas de marketing, contendo informações sobre perfil de clientes, compras e interações com campanhas.

---

## 🎯 Objetivos

* Avaliar a performance de campanhas de marketing
* Analisar taxa de conversão de clientes
* Identificar padrões de comportamento de compra
* Segmentar clientes para apoio a decisões estratégicas

---

## 🧱 Estrutura do Projeto

### 🔹 Preparação de Dados

* Tratamento e limpeza no Power Query
* Padronização de variáveis
* Criação de colunas derivadas

---

### 🔹 Métricas (DAX)

* Taxa de Conversão
* Receita Total
* Ticket Médio
* Total de Clientes
* Clientes convertidos

---

### 🔹 Dashboard

#### 📄 Página 1 — Resumo Executivo

* KPIs principais
* Conversão geral
* Receita por campanha
* Segmentação de clientes

#### 📄 Página 2 — Análise Detalhada

* Performance por campanha
* Comportamento por canal (web, loja, catálogo)
* Perfil de clientes (renda, idade)
* Tabela detalhada

---

## 📊 Principais Insights

* Identificação das campanhas com maior taxa de conversão
* Segmentos de clientes mais engajados
* Diferença de comportamento entre canais de compra
* Relação entre renda e consumo

---

## 🛠️ Tecnologias Utilizadas
